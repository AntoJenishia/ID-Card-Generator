# ID Card Generator System (Java + MySQL)

A client-server based Java Swing application to generate and manage student ID cards.

## Features
- Add new student records
- Auto-save ID card as PNG
- Auto-copy student photos to `/photos/`
- View all students in database
- Secure DB connection via `.env`
- JDBC + Socket + Swing Architecture

## Technologies
- Java 17
- MySQL 8
- JDBC
- Java Swing (Frontend)
- Socket Programming

## How to Run
1. Start MySQL server and ensure database `IDCard` exists.
2. Run `IDCardServer.java` to start backend.
3. Run `IDCardClient.java` to open GUI.
4. Enter details → Choose photo → Generate ID.
